# Algoritmos e Estruturas de Dados – INATEL (C++)

Este repositório reúne os exercícios, implementações e práticas desenvolvidas na disciplina de C02 - Algoritmos e Estruturas de Dados durante a graduação em Engenharia de Computação no INATEL.

## 📌 Objetivo

Consolidar fundamentos de lógica de programação e estruturação de algoritmos utilizando C++, com foco em:

- Estruturas condicionais
- Estruturas de repetição
- Funções
- Vetores e matrizes
- Manipulação básica de dados
- Organização e boas práticas de código

## 🛠 Tecnologias

- C++
- Compilador padrão (falcon)
- VS Code

## 📚 Observação

Este repositório tem caráter acadêmico e acompanha minha evolução na disciplina.
